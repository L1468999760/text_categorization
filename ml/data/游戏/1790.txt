首款DX11游戏大作《尘埃2》驶入中国国内资深游戏运营机构－－北京中电博亚科技有限公司近日宣布，继《科林麦克雷拉力赛：尘埃》(ColinMcRae Dirt)、《超级房车赛3》(Toca Race Driver 3)之后，英国著名赛车游戏厂商Codemasters公司的最新力作《科林麦克雷拉力赛：尘埃2》(Dirt2)也将引入国内，并将在全国展开大规模的市场宣传与推广活动，包括与AMD公司的ATI Radeon HD5000系列显卡合作推广。科林麦克雷系列在玩家心目中一直是越野拉力赛的代名词，全球总销量已超过10,000,000套，而系列的最新成员《科林麦克雷拉力赛：尘埃2》作为全球首款支持DX11视觉特效的AAA级赛车游戏巨作，更是将3D视频显示效果发挥到了极致。据悉，Codemasters自行研发的EGO引擎将整体部署DirectX 11技术，支持图形多线程、硬件Tessellation以及SM5.0等新特性，被喻为目前最棒的赛车游戏。AMD公司的ATI Radeon HD5000系列显卡是其最新一代显卡产品，也是全球最先支持DirectX11的系列显卡产品，能够完全的将DX11游戏中的显示效果发挥出来。而在国内市场方面，北京中电博亚科技有限公司此次将配合AMD共同开启ATI Radeon HD5000系列显卡与《尘埃2》的捆绑销售计划，可谓是打造游戏与硬件并存的梦幻组合。很显然，DX11硬件+游戏捆绑计划所带来的并不是一款简单的促销活动，而是在推动着整个DirectX 11技术的发展，甚至可以称得上是显卡以及游戏领域发展的风向标。《科林麦克雷拉力赛：尘埃2》中包括了伦敦、洛杉矶和中国在内9个地区的91条赛道，35辆不同种类的三菱、福特、斯巴鲁及通用悍马等越野拉力劲车也将悉数登场。游戏中的比赛模式共分8种，既有传统的场地拉力、长距离越野，也有让人耳目一新的击牌过关、占路为王等，为玩家一一展现出次世代赛车游戏的风貌。对于全球首款支持Windows7和DX11特性的游戏，只有强劲性能的配置才能彰显其与众不同的魅力，但是DX9同样得到支持。最低配置需求：系统： Windows XP, Vista or Windows 7处理器： AMD Athlon 64 3400+内存： 1GB (2GB for Windows Vista)硬盘： 10GB显卡： ATI Radeon X1550推荐配置需求：系统： Windows Vista 或 Windows 7处理器： AMD Athlon II 及以上CPU产品内存： 2GB硬盘： 10GB显卡： ATI Radeon HD 5600/5700/5800 系列精彩游戏，精彩人生，尽在中电博亚。更多游戏产品敬请访问官方网站中电博亚官方推荐玩家选用AMD公司支持DirectX 11的ATI Radeon HD 5000系列显卡，以获得最佳游戏体验。
