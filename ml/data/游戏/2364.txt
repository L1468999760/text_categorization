微软下月将发布官方在线游戏下载商店微软今天宣布了一个新的专门提供游戏的在线商店Games for Windows Marketplace，游戏玩家可以直接在PC上下载游戏，该商店将在11月15日上线。微软承诺，这个游戏商店针对下载速度进行了优化，只需简单的几键点击就能轻松购买下载游戏。微软PC和手机游戏高级主管Kevin Unangst表示，这是为PC游戏玩家提供的一种端到端的游戏商店，该商店将整合现有的Xbox Live在线平台游戏服务和Windows Live。玩家可以随时随地购买并下载游戏，因为基于Web，所以用户还可以将购买的游戏重新下载到其它计算机上。在Games for Windows Marketplace中，搜索游戏变得十分简单，你可以直接按名称搜索，也可以按类别搜索。Games for Windows Marketplace上线时将提供来自微软游戏工作室、2K Games、Capcom、Rockstar和华纳兄弟的100多款游戏大作，包括《神鬼寓言：失落之章》、《侠盗猎车3》等经典游戏，诸如《Lego Universe》、《CarneyVale》的游戏新作也会在上市的同时出现在Games for Windows Marketplace里。
